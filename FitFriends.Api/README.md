# Проект "FitFriends"

Приветствуем вас в проекте "FitFriends"! 
FitFriends — это онлайн площадка для поиска тренировок и их создания. 
Веб-приложение (сайт) функционирует как смесь соцсети и биржи объявлений. 
Тренеры создают тренировки, а пользователи могут покупать их и заниматься по ним, 
приглашать других пользователей на совместные тренировки, а тренеров просить провести индивидуальную тренировку.

## Запуск проекта с использованием Docker Compose

Выберите запускаемый проект docker-compose. Запустите проект. Дождитесь пока приложение скачает необходимые images и развернёт контейнер.

## Инициализация базы данных

В корне src лежит файл script.sql. Запустите его прямо со студии.

Имя сервера можно взять со строки подключения в appsettings.json (host.docker.internal,5050)
